# AtliQ-Hospitality-Analytics-Dashboard-Power-BI

**📌 Project Overview
**
* Developed an interactive Power BI dashboard to analyze hotel performance across multiple properties and cities.
* Created a centralized reporting solution for monitoring revenue, occupancy, booking trends and operational KPIs.
* Enabled data-driven decision-making through interactive visualizations and performance tracking.

🎯 Business Problem

* AtliQ Hotels operates multiple properties across different cities and room categories.
* Management required a centralized dashboard to:

  * Track revenue performance.
  * Monitor occupancy and booking trends.
  * Analyze customer booking behavior.
  * Evaluate key hospitality KPIs.
  * Identify opportunities to improve business performance.

🛠️ Tools & Technologies Used

* Power BI Desktop
* Power Query
* MySQL
* DAX (Data Analysis Expressions)
* Data Modeling
* Microsoft Excel

📊 Key Performance Indicators (KPIs)

* Revenue
* RevPAR (Revenue per Available Room)
* ADR (Average Daily Rate)
* Occupancy %
* Realization %
* DSRN (Daily Sellable Room Nights)
* DBRN (Daily Booked Room Nights)
* DURN (Daily Utilized Room Nights)

📈 Dashboard Features

* Revenue Analysis by City
* Weekly Revenue Trend Analysis
* Occupancy Performance Monitoring
* Booking Platform Analysis
* Room Category Performance Analysis
* Week-over-Week KPI Comparison
* Property-Level Performance Insights

🔍 Key Analysis Areas

* Revenue performance across cities and hotel properties.
* Occupancy, ADR, RevPAR and Realization rate analysis.
* Booking trends across platforms and room categories.
* Week-over-week KPI performance tracking.
* Property-level performance comparison.
* Booking cancellation analysis and revenue impact assessment.
* Customer demand analysis across different room categories.

💡 Business Recommendations

* Focus marketing efforts on high-performing markets and properties.
* Improve occupancy in underperforming properties through targeted campaigns.
* Strengthen relationships with effective booking platforms.
* Apply demand-based pricing strategies to optimize revenue.
* Monitor cancellation trends to improve realization rates.
* Use KPI tracking to support operational and strategic decision-making.

🚀 Project Outcome

* Built a centralized hospitality performance dashboard.
* Improved visibility into key business and operational metrics.
* Enabled performance comparison across properties, cities and room categories.
* Supported data-driven decision-making through interactive analytics.
